# AI Resume Analyzer

A monorepo for the AI Resume Analyzer project.

## Structure
- `/frontend` - Nuxt 3, Vue, Tailwind CSS
- `/backend` - FastAPI, SQLAlchemy, PostgreSQL

## Backend Setup
1. `cd backend`
2. `python3 -m venv venv`
3. `source venv/bin/activate`
4. `pip install -r requirements.txt`

## Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Database
Run `docker-compose up -d` to start the PostgreSQL database.
